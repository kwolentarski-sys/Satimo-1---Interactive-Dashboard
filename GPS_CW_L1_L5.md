**GPS CW L1 L5**

**Test Description:**

* A continuous wave (non-modulated) signal at a level of \-90 dBm in the quiet zone is used to establish a pattern for the DUT.  
* Qualcomm Tools are used (QMSL) to report the carrier-over-noise level for each test position.  
* A WiFi router is used inside the chamber as a communication path from the DUT to the chamber control pc.
